# React Web Application SBA-320

This is a front end react project. It is a Harry Potter fact site. The home page is filled with all the Harry Potter character you know & love. When you click on the card it flips over & give you information on the characters. The navigation bar has links to the houses, spells, & books that you would be able to flip over & get more information on.
